## 코드 작성 규칙


1. 주석
      - Class 위: @since, @author, 설명
      - Method 위 : 설명, (argument, return)
      - Class를 만들었을때 어떤식으로 구상 하고싶은지 주석처리하여 매모를 하고, 기반으로 작성하면 좋음
1. Package
      - 본인이 짠 Class와 타인의 Class 를 구별하기위해, 
		com.[본인의 닉네임 혹은 이니셜].[대분류] 로 구성 할 것.
